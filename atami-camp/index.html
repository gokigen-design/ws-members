コーデックスって？

OpenAIが出しているAIコーディングエージェントです。Claude Codeと同じく、ターミナルに話しかけてLP・サイト・ツールを作れます。画像生成と組み合わせやすく、デザイン寄りのものを作るときに相性がいいのが特徴です。

ダウンロード方法

  1. ターミナルを開く（Macは「ターミナル」アプリ）
  2. npm install -g @openai/codex を実行（※Node.jsが必要）
  3. codex と打って起動。初回はChatGPTアカウントまたはAPIキーでログイン

基本の使い方

  • 作りたいものを、日本語で話しかけるだけ（Claude Codeと同じ感覚）
  • 「こんなLPを作って」「ここをこう直して」で進めていける
  • うまくいかない時は、イメージや参考画像を見せると伝わりやすい

こんな使い方ができます

  • デザイン性のあるLP制作（写真・ビジュアル多めも得意）
  • 画像生成と組み合わせた提案資料づくり
  • 既存コードの修正・整理
インストール手順は変わることがあるので、当日いっしょに最新版で確認します。うまくいかなくても大丈夫、その場でサポートします。
